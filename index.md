# 📜 Visa Tips

1. [Document Checklist](https://www.vfsglobal.com/one-pager/switzerland/india/english/pdf/Checklist_Tourist-EN-2025.pdf)
2. Get a **6-month bank statement** before appointment.
3. Old passport usually not required.
4. Request **leave letter + employment letter**.
5. Visa process took ~1.5 hours.
6. Use **current address** on application form.
7. Visa approved within **5 days**.
